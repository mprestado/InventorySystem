==========================================================================
 Houseware Inventory Management System — Requirements
==========================================================================
 NOTE: This is a Laravel (PHP) project. PHP dependencies are managed by
 Composer (composer.json / composer.lock) and JS tooling by npm
 (package.json). This file is a human-readable summary of everything
 needed to install and run the project.

 Install PHP packages with:   composer install
 Install JS packages with:    npm install   (optional — CDN assets used by default)
--------------------------------------------------------------------------

# --- System / Software Prerequisites -------------------------------------
PHP >= 8.2
Composer >= 2.x
MySQL >= 8.0    (or MariaDB >= 10.4)
Node.js >= 18   (optional, only for a Vite asset build)
npm >= 9        (optional)
XAMPP (recommended on Windows — bundles PHP + MySQL)
